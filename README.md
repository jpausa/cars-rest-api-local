# Simple Rest Api about cars and brands.

## Description

A simple Rest Api that exposes a CRUD about cars and brands. The data is seeded locally. No datbases used.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Loading the data

`POST http://localhost:3000/seed`

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
